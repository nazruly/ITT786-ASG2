How to run Java program:
Java version was written using Netbean version 8.0.2 with JDK1.8
All files in this folder were copied from the project folder.
Please run by individual java file. Run Server.java first then you can 
run multiple instance of Client.java. 

How to run Perl version:
Perl version was written using normal text editor.
The script located in a folder named Perl.
Both SSLTLS-Server.pl and SSLTLS-Client.pl using library IO:Socket:SSL.
This library can be installed using CPAN command.
Please run SSLTLS-Server.pl first and then SSLTLS-Client.pl.
Server script only accept one connection at one time.
